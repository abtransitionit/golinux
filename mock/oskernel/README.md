# Terminlogy
|name|type|extension|comment|
|-|-|-|-|
|osk|acro|**OS** **K**ernel|
|oskm|acro|**OS** **K**ernel **m**odule|
|oskp|acro|**OS** **K**ernel **p**arameter|

# Purpose
`osk` provides a generic way to manage linux kernel modules and modules parameters. It works on major Linux families:

* **Debian-based** (APT)
* **RHEL-based** (DNF)
