# Terminlogy

|osse|acro|**OS** **S**ervice **S**tart|
|osss|acro|**OS** **K**ernel **E**nable|

# Purpose
`osservice` provides a generic way to manage linux native services. It works on major Linux families:

* **Debian-based** (APT)
* **RHEL-based** (DNF)
